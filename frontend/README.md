# AllYourDocs - Ultimate Document Processing Toolkit

A modern, client-side web application for all your document processing needs. Built with Next.js, React, and Tailwind CSS.

## 🚀 Live Demo
[https://allyourdocs.vercel.app](https://allyourdocs.vercel.app)

## ✨ Features

### **PDF Tools**
- 📄 **Merge PDFs** - Combine multiple PDF files into a single document
- 🔄 **PDF to Word** - Convert PDF files to editable Word documents
- 📝 **Word to PDF** - Convert Word documents to PDF format
- ✂️ **Split PDF** - Split PDF files by page ranges
- 🖼️ **Images to PDF** - Convert multiple images to a PDF document
- 📦 **Compress PDF** - Reduce PDF file size without losing quality
- 🔤 **PDF to Text** - Extract text from PDF documents
- 🔀 **Rearrange PDF** - Reorder, rotate, or delete PDF pages

### **Image Tools**
- 👁️ **Image to Text (OCR)** - Extract text from images using OCR

### **General Features**
- 🔒 **100% Client-Side Processing** - Files never leave your browser
- 🚫 **No Watermarks** - Clean outputs without branding
- ⚡ **Fast Processing** - Real-time document manipulation
- 📱 **Responsive Design** - Works on all devices
- 🎨 **Modern UI** - Clean, intuitive interface
- 📧 **Feedback System** - Direct communication with developers
